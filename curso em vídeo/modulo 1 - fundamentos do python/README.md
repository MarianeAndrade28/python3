# Python.
O Python se tornou uma das linguagens de programação mais populares do mundo nos últimos anos. Isso se deve, principalmente, à sua versatilidade. Funciona tanto para o aprendizado de máquina, construção de sites, quanto para a automação de tarefas e testes de software. Além disso, sua proximidade com a linguagem humana faz com que muitas pessoas o utilizem, seja desenvolvedor ou quem necessariamente faz parte desse mercado.
Uma linguagem de programação de uso geral, ou seja, uma linguagem que funciona para criar uma variedade de aplicações.

**7 motivos para aprender:**
* Possui uma sintaxe simples;
* É multiplataforma e de código aberto;
* É versátil;
* Tem uma comunidade fiel e ativa;
* É utilizado por grandes empresas;
* É o mais popular na Ciência de Dados;
* Está em alta no mercado de trabalho.
